Конфигурация

Эта страница описывает, что можно изменить в конфигурации приложения.

Тема

Задайте светлую или темную тему, или сделайте так, чтобы Tagger соответствовал теме вашей системы.

Запомнить последнюю открытую библиотеку

Нужно ли запоминать ранее использовавшуюся музыкальную библиотеку и открывать ее при следующем запуске Tagger.

Включать подпапки

Сканировать ли музыкальные файлы в подкаталогах музыкальной папки.

Сортировать файлы по

Свойство, используемое при сортировке и отображении музыкальных файлов. Это может быть один из следующих вариантов:

  • Имя файла

  • Путь к файлу

  • Название

  • Artist

  • Альбом

  • Year

  • Дорожка

  • Жанр

Сохранение временной метки модификации

Следует ли не обновлять временную метку записи (модификации) файла при сохранении его метки.

Перезапись тегов с помощью MusicBrainz

Следует ли перезаписывать существующие метаданные тегов данными, полученными из MusicBrainz. Если отключено, то будут заполняться только пустые свойства тегов.

Переписать обложку альбома с помощью MusicBrainz

Нужно ли перезаписывать существующие обложки альбомов обложками, найденными из MusicBrainz.

Перезапись текстов с помощью веб-сервиса

Следует ли перезаписывать имеющиеся в теге текстовые данные данными из Интернета.

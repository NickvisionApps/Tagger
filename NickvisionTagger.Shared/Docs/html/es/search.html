Búsqueda avanzada

En esta página se explica cómo usar la búsqueda avanzada en Tagger.

La búsqueda avanzada es una potente función proporcionada por Tagger que permite a los usuarios buscar determinados valores en el contenido de las etiquetas de los archivos, usando una potente sintaxis basada en etiquetas:

!prop1="valor1";prop2="valor2"

Donde prop1, prop2 son propiedades de etiqueta válidas y value1, value2 son los valores a buscar entre comillas. Cada propiedad está separada por un punto y coma. Observe que la última propiedad no termina con punto y coma.

Propiedades válidas

  • nombredearchivo

  • título

  • artista

  • álbum

  • año

  • pista

  • pistatotal

  • artistadelálbum

  • género

  • comentario

  • pulsacionesporminuto (ppm)

  • compositor

  • descripción

  • discnumber

  • disctotal

  • editor

  • publishingdate

  • personalizado

Las propiedades distinguen entre mayúsculas y minúsculas.

Comprobación sintáctica

Si la sintaxis de su cadena es válida, el cuadro de texto se volverá verde y filtrará la lista con su búsqueda. Si la sintaxis de la cadena no es válida, el cuadro de texto se volverá rojo y no filtrará la lista.

Ejemplos

!artista=""

Esta cadena de búsqueda filtrará el cuadro de lista para que contenga archivos de música cuyo artista esté vacío.

!género="";año="2022"

Esta cadena de búsqueda filtrará el cuadro de lista para que contenga archivos de música cuyo género esté vacío y cuyo año sea 2022 (las propiedades año y pista se validarán si la cadena de valores es un número).

!título="";artista="bob"

Esta cadena de búsqueda filtrará el cuadro de lista para que contenga archivos de música cuyo título esté vacío y cuyo artista sea bob.

!personalizado="mbrd"

Esta cadena de búsqueda filtrará el cuadro de lista para que contenga archivos de música que contengan una propiedad personalizada con el nombre mbrd.

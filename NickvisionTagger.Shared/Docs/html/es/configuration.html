Configuración

Esta página describe lo que puede cambiar en la configuración de la aplicación.

Tema

Establezca un tema claro u oscuro, o haga que Tagger siga el tema de su sistema.

Recordar última carpeta abierta

Recordar o no la carpeta de música usada anteriormente y volver a abrirla cuando Tagger se inicie la próxima vez.

Incluir subcarpetas

Buscar o no archivos de música en subdirectorios de una carpeta de música.

Ordenar archivos por

La propiedad que se usará para ordenar y mostrar los archivos de música. Puede ser una de las siguientes opciones:

  • Nombre de archivo

  • Título

  • Pista

  • Ruta del archivo

  • Álbum

  • Género

Conservar fecha de modificación

Si actualizar o no la marca de tiempo de escritura (modificado) de un archivo cuando se guarda su etiqueta.

Sobreescribir etiqueta con MusicBrainz

Si sobreescribir o no los metadatos de las etiquetas existentes con los datos encontrados en MusicBrainz. Si se desactiva, sólo se rellenarán las propiedades de etiqueta en blanco.

Sobreescribir la portada del álbum con MusicBrainz

Sobreescribir o no las portadas de los álbumes existentes con las encontradas en MusicBrainz.

Clave API de usuario de AcoustId

Una clave API de usuario de AcoustId que se usará para identificar archivos de música y buscar información sobre canciones.

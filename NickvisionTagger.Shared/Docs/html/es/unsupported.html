Archivos no compatibles

Esta página explica los archivos de música que actualmente no son compatibles con Tagger.

Tagger usa atldotnet como base para leer y escribir metadatos.

No se pueden aplicar los cambios - Permiso denegado

Existe un problema conocido de la versión flatpak de la aplicación que se ejecuta en sistemas basados en Debian en el que no se pueden guardar archivos debido a un error de "permiso denegado".

Por favor, mira esto para corregirlo: https://github.com/NickvisionApps/Tagger/issues/174

Por favor, confirma que recibe el error "permiso denegado" antes de intentarlo.

Ejecuta Tagger en el terminal mediante flatpak run org.nickvision.tagger, luego intenta aplicar los cambios de nuevo, y encontrarás un mensaje de error en el terminal.

Si no es un error de "permiso denegado", por favor reporta un nuevo problema.

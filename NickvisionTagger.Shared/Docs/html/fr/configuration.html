Configuration

Cette page décrit ce que vous pouvez modifier dans la configuration de l’application.

Thème

Définissez un thème clair ou sombre, ou commandez à Tagger de suivre votre thème système.

Mémoriser le dernier dossier ouvert

Indique s’il faut mémoriser ou non le dossier de musique précédemment utilisé et le rouvrir par défaut au prochain lancement de Tagger.

Inclure les sous-dossiers

Indique s’il faut ou non inspecter les sous-répertoires d’un dossier de musique pour y rechercher des fichiers musicaux.

Trier les fichiers par

Il s’agit de la propriété utilisée pour trier et afficher les fichiers musicaux. Cela peut être une des options suivantes :

  • Nom du fichier

  • Titre

  • Piste

  • Chemin d’accès du fichier

  • Album

  • Genre

Conserver l’horodatage de modification

Indique s’il faut ou non mettre à jour l’horodatage de la dernière écriture (modification) d’un fichier lorsqu’une balise est enregistrée.

Écraser la balise avec MusicBrainz

Indique s’il faut ou non écraser les métadonnées des balises existantes avec les données récupérées depuis MusicBrainz. Si cette option est désactivée, seules les propriétés des balises laissées vides seront remplies.

Écraser la couverture d’album avec MusicBrainz

Indique s’il faut ou non remplacer la couverture de l’album existante avec celle récupérée depuis MusicBrainz.

Remplacer les paroles avec le service web

Indique s’il faut ou non remplacer les paroles existantes d’une balise de données avec des données récupérées depuis le web.

Clé utilisateur d’API AcoustId

Une clé d’API AcoustId utilisateur à utiliser pour récupérer l’empreinte des fichiers musicaux et regarder les informations du morceau.

Configurazione

Questa pagina descrive quali configurazioni dell'applicazione puoi modificare.

Tema

Imposta il tema chiaro o scuro, oppure utilizza il tema di sistema per Tagger .

Memorizza l'ultima cartella aperta

Se memorizzare o meno l'ultima cartella utilizzata ed riaprirla al prossimo avvio di Tagger.

Includi le sottocartelle

Se cercare o meno musica all'interno delle sottocartelle.

Ordina file per

La proprietà usata per ordinare e visualizzare file musicali. Può essere una delle seguenti opzioni:

  • Nome del file

  • Titolo

  • Traccia

  • Percorso del file

  • Album

  • Genere

Mantieni l'istante temporale di modifica

Se aggiornare o meno l'istante temporale di scrittura quando un tag è stato modificato.

Sovrascrivi i tag con With MusicBrainz

Se sovrascrivere i tag esistenti con i dati provenienti da MusicBrainz. Se disabilitato, verranno riempiti solo i campi lasciati vuoti.

Sovrascrivi la copertina con MusicBrainz

Se sovrascrivere o meno la copertina esistente con quella trovata su MusicBrainz.

Chiave delle API di AcoustId dell'utente

Una chiave per le API di AcoustId da usare per cercare le informazioni relative ai brani.

Unsupported File

This page explains music files that are currently unsupported by Tagger.

Tagger uses atldotnet for its backend to read and write metadata. However, there are some currently known file limitations.

OGG With FLAC Tag

OGG files that use an embedded FLAC tag will be opened in read-only mode.

Issue tracking writing support for ogg/flac

OPUS With Cover Art

Certain OPUS files that contain cover art are unable to be saved.

The file is readable however cover art will not be loaded and any modifications cannot be saved. Other OPUS files may work correctly without issue.

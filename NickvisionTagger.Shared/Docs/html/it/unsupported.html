Unsupported Files

This page explains music files that are currently unsupported by Tagger.

Tagger uses atldotnet for its backend to read and write metadata.

Unable To Apply Changes - Permission Denied

There is a known issue of the flatpak version of the app running on Debian-based systems where you are unable to save files due to a "permission denied" error.

Please see this for the fix: https://github.com/NickvisionApps/Tagger/issues/174

Please confirm that you are receiving the "permission denied" error before trying this.

Run Tagger in the terminal via flatpak run org.nickvision.tagger, then try to apply changes again, and you will find an error message in the terminal.

If it is not a "permission denied" error, please report a new issue.
